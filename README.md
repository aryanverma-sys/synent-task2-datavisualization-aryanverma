# Task 2: Data Visualization - Iris Feature Analysis

##  Problem Statement
The objective of this project is to explore and visualize statistical patterns, structural distributions, and feature relationships within the classic Iris flower dataset using graphical representations.

##  Dataset Details
* **Source:** Built-in Seaborn datasets (Iris Dataset)
* **Features Explored:** Sepal Length, Sepal Width, Petal Length, Petal Width, and Species (Setosa, Versicolor, Virginica).

##  Step-by-Step Approach
1. **Environment Setup:** Imported `matplotlib` and `seaborn` plotting frameworks within Jupyter/Colab environments.
2. **Distribution Tracking:** Created an automated histogram with a Kernel Density Estimate (KDE) curve to chart the statistical variance of sepal width.
3. **Categorical Analysis:** Generated bar charts evaluating mean trait boundaries across different floral classes.
4. **Bivariate Comparison:** Created color-coded multi-class scatter plots to isolate operational feature clusters between different species.

##  Key Results & Insights
* **Cluster Separation:** The scatter plots reveal a clear, distinct spatial boundary separating the *Setosa* species from *Versicolor* and *Virginica* based purely on structural measurements.
* **Trait Margins:** *Setosa* displays significantly shorter petal structures on average compared to the other two variants.
